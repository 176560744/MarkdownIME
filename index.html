Imagine a rich textbox without any button, it's easy if you try.

MarkdownIME provides a fresh way to write well-formatted text, by typing markdownized stuff. It will speed up writing, help you get rid of the small formatting buttons.

Works like a charm on mobile devices, too!

Features

MarkdownIME supports many Markdown syntax. Just type in Markdown style, and then press Enter.

Refer: Markdown Basic

Supported Markdown Syntax

Span Elements

  • Links
  • Emphasis: **Bold** & *Italy*
  • ~~Strikethrough~~
  • `In-line Code`
  • Images
  • emoji :-) [1]
  • Tex formulas: $ E=mc^2 $ [2]
  1. Emoji feature is provided as an add-on. System loads it automatically. Works better with twemoji.
  2. Tex formulas feature is provided as an add-on, which will NOT load automatically. Read the source code and the comments.

Block Elements

  • Headers (beginning with#)
  • Horizontal Line ---
  • List & Nested list
  • Blockquote & Nested blockquote
  • Code Block

Basic table

MarkdownIME support creating table with | column | header | texts | and more.

When editing a cell, press Enter key to insert a row.

Press Enter key twice to end the table and start a new line of text.

Besides, MarkdownIME enhances the key-bindings when editing table. You can move the cursor from cell to cell swiftly, with Tab, Shift + Tab, Up Arrow and Down Arrow keys.

Supported Editor

  • Vanilla contenteditable elements - browsers solve everything.
  • TinyMCE - now typing on EverNote and others is easier.
