Features

MarkdownIME supports many Markdown syntax. Just type in Markdown style, and then press Enter.

Refer: Markdown Basic

Supported Markdown Syntax

Span Elements

  • Links
  • Emphasis: **Bold** & *Italy*
  • ~~Strikethrough~~
  • `In-line Code`
  • Images

Block Elements

  • Headers (beginning with#)
  • Horizontal Line ---
  • List & Nested list
  • Blockquote & Nested blockquote
  • Code Block

Supported Editor

  • Vanilla contenteditable elements - browsers solve everything.
  • TinyMCE - now typing on EverNote and others is easier.
