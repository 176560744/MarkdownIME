Introducing

MarkdownIME is a "plugin" for the rich editors on web pages.

Quickstart

For users who enjoy markdown

Load MarkdownIME

Drag this bookmarklet to your bookmark bar. When you feel bad, click it and everything goes better.

For developer

This plugin is still unstable. If you want to try, here is some simple APIs.

  • MarkdownIME.scan(window) scans all the content-editable elements (aka. editors) and enhances them.
  • MarkdownIME.prepare(window, editor) enhances one editor.

Features

Supported Markdown Syntax

Span Elements

  • Links
  • Emphasis
  • In-line Code
  • Images

Block Elements

  • Headers (beginning with#)
  • Horizontal
  • Lists (nested list is not supported yet)

NOTICE: Backslash Escapes is not supported yet.

Supported Editor

  • Vanilla contenteditable elements - browsers solve everything.
  • TinyMCE - now typing on EverNote and others is easier. Try out »

TinyMCE is partially supported!

But you need some trick.

  1. Load the editor with tinymce.init.
  2. Find out the contentWindow and call MarkdownIME.prepare(the_window, the_contenteditable_element)
